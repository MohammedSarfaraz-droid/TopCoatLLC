

# Topcoat LLC — Premium Flooring & Decorative Concrete Website

## Brand & Design System
- **Dark mode dominant** with charcoal/black backgrounds
- **Metallic copper/brushed gold** accent colors throughout
- **Typography**: Elegant serif for headings (Playfair Display), modern sans-serif for body (Inter)
- **Glassmorphism** panels and cards where appropriate
- **Smooth animations**: fade-in on scroll, parallax effects, hover micro-interactions
- **Sticky header** that transitions from transparent to solid on scroll

All images will be generated using the AI image generation API — no placeholders. We'll generate 10+ ultra-realistic architectural photography images covering epoxy showrooms, terrazzo interiors, garage coatings, stamped concrete patios, building facades, artisan action shots, and more.
